\subsubsection{Detailed Description}
The \hyperlink{class_data_packet}{DataPacket} stores all sampled data from every connected device. The \hyperlink{class_data_packet}{DataPacket} contains the samples from all connected devices in ascending order (by their signal type). It stores acquired samples in a vector and is able to build a raw representation including needed header information of its stored samples.
